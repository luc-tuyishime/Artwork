# Art API

A frontend App that display beautiful Arts.

## Tools Used

### Languages
```
Javascript - typescript
```

### Library

```
React
```

### Style Guide
```
Eslint - prettier 
```

## Installing
To run the app, you will need to install *NodeJS* on your Operating System if you don't have it installed.

[Install NodeJS](https://nodejs.org/en/)

After run 

```
yarn
```

## Run the server
```
> yarn start
```
## Run the test
```
> yarn test
```
