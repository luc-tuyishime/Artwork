## Installing

To run the app, you will need to install _NodeJS_ on your Operating System if you don't have it installed.

[Install NodeJS](https://nodejs.org/en/)

After run

```
yarn
```

## Run the server

```
> yarn start
```

## Run the test

```
> yarn test
```
